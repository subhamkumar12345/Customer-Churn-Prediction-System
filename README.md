# 📊 Customer Churn Prediction Using Machine Learning

## 📌 Project Overview

This project focuses on predicting customer churn using Machine Learning techniques. Customer churn prediction helps businesses identify customers who are likely to leave and enables proactive retention strategies.

The project includes data cleaning, feature engineering, exploratory data analysis (EDA), model building, hyperparameter tuning, and performance evaluation.

---

## 🎯 Business Problem

Customer retention is significantly more cost-effective than acquiring new customers. Businesses need to identify customers at risk of churn and take preventive actions.

The objective of this project is to:

- Predict customer churn accurately
- Identify factors contributing to customer attrition
- Improve customer retention strategies
- Support data-driven business decisions

## 📂 Dataset Features

The dataset contains customer-related information such as:

- Internet Option
- Days Since Last Login
- Average Time Spent
- Points in Wallet
- Number of Referrals
- Complaint Status
- Feedback
- Churn Status (Target Variable)

## 🔍 Project Workflow

### 1. Data Preprocessing

- Handled missing values
- Removed irrelevant features
- Encoded categorical variables
- Prepared dataset for modeling

### 2. Exploratory Data Analysis (EDA)

Performed analysis on:

- Customer Demographics
- Membership Categories
- Internet Usage Patterns
- Referral Behavior
- Complaint Trends
- Customer Feedback Analysis
- Churn Distribution

### 3. Feature Engineering

- One-Hot Encoding
- Missing Value Treatment
- Feature Selection
- Data Transformation

### 4. Machine Learning Models

Implemented and compared:

- Random Forest Classifier
- XGBoost Classifier

### 5. Hyperparameter Tuning

Applied:

- RandomizedSearchCV

to improve model performance.

### 6. Model Evaluation

Performance evaluated using:

- Accuracy Score
- Confusion Matrix
- Classification Report
- Precision
- Recall
- F1 Score

---

## 📊 Key Insights

- Membership category strongly influences churn behavior.
- Customer complaints increase churn probability.
- Internet service preferences impact customer retention.
- Positive customer feedback correlates with lower churn rates.
- Referral-based customers tend to be more loyal.

---

## 🚀 Results

The XGBoost model achieved strong predictive performance and successfully identified customers at risk of churn.

The model can help businesses proactively engage customers and reduce customer attrition.
